A secure bank-backed auction platform where users must deposit funds before placing bids, ensuring trusted and verified transactions.

# Features
- Secure bidding system with fund reservation
- Wallet management (available balance, funds on hold, pending deposits)
- Automatic fund hold and release logic
- Admin dashboard for user verification and deposit approvals
- Real-time auction monitoring
- User authentication and secure access

# Tech Stack
- Frontend: React.js, HTML5, CSS3, JavaScript
- Backend: Django
- Database: MySQL
